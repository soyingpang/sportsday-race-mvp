# 親子遊戲日（本機版 / 不用 GitHub / 不用同步）

本版依你的最新流程調整：

1. **下載賽程表（CSV）與計分總表（CSV）**：都會按「場次/年級」排序  
2. **看板不再顯示賽程表**：只顯示「黎緊三隊」＋「1/2 年級 Top10（3 遊戲總時間）」  
3. **計分員不需入分**：收齊手寫分數後，由後台一次輸入  
4. **排名只看總時間**：每位參賽者 = 3 個遊戲秒數總和；分別取一年級 Top10、二年級 Top10

---

## 專案結構
- `index.html`：後台（匯入名單、建立賽程、指定目前場次、輸入 3 遊戲時間、下載 CSV）
- `board.html`：看板（黎緊三隊 + 1/2 年級 Top10）
- `score.html`：已停用（保留說明頁）
- `assets/`：前端程式（零依賴）
- `data/participants.sample.csv`：範例名單

---

## 本機開啟方式（建議）
因為有用到 `fetch('./data/participants.sample.csv')`，建議用本機靜態伺服器：

```bash
python -m http.server 8000
```

開啟：
- 後台：`http://localhost:8000/index.html`
- 看板：`http://localhost:8000/board.html`

---

## CSV 欄位
`id,class,no,name,present`

- `class` 例如：1A / 1B / 2A / 2B
- `present`：1/0 或 true/false

---

## 操作流程（活動當日）
1. 後台匯入名單（CSV）
2. 後台建立賽程（可一鍵自動排完），並用「設為」指定目前場次
3. 看板會顯示：黎緊三隊（請準備）
4. 活動結束後，把手寫分數表交回後台 → 後台輸入三個遊戲秒數
5. 看板自動顯示：一年級/二年級 Top10（總時間）
6. 下載：
   - 賽程表 CSV（按場次排序）
   - 計分總表 CSV（含名次/總時間）
